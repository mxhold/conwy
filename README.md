# Conway

This gem plays [Conway's Game of
Life](https://en.wikipedia.org/wiki/Conway's_Game_of_Life).

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'conway'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install conway

## Usage

```
$ conway < echo "010\n010\n010"
000
111
000
```
